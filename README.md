# MonadMetrics

MonadMetrics is a web application for monitoring and visualizing metrics from the Monad blockchain testnet. The application provides real-time data about blocks, transactions, and network performance.

## Features

- Real-time blockchain metrics dashboard
- Block explorer with detailed information
- Transaction history and analysis
- Performance comparison with other blockchain networks

## Tech Stack

- [Remix](https://remix.run/) - Full stack web framework
- [TailwindCSS](https://tailwindcss.com/) - For styling
- [Ethers.js](https://docs.ethers.org/) - Ethereum library
- [Farcaster Frame SDK](https://github.com/farcasterxyz/frame-sdk) - For frame interactions

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/BluOwn/MonadMetrics.git
cd MonadMetrics
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`
